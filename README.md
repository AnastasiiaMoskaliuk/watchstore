# ⌚ WatchStore: Time in Every Detail


## ✨ Про проєкт
**WatchStore** — це не просто інтернет-магазин, це простір, де естетика годинникового мистецтва зустрічається з сучасними веб-технологіями. Проєкт створений як повноцінне Fullstack-рішення для демонстрації та продажу преміальних аксесуарів.

> "Ми не просто продаємо годинники, ми допомагаємо вам керувати вашим часом зі смаком."

---

## 🎨 Особливості дизайну та UX
* **Minimalist UI:** Чистий інтерфейс, який не відволікає від головного — вибору ідеального годинника.
* **Fully Responsive:** Сайт адаптований під будь-які екрани: від великих моніторів до смартфонів у вашій кишені.
* **Smooth Navigation:** Плавні переходи та інтуїтивно зрозумілий шлях користувача від головної сторінки до кошика.

---

## 🛠 Технологічне серце (Stack)

### 💻 Frontend (Клієнтська частина)
* **React.js:** Побудова динамічного та швидкого інтерфейсу.
* **Modern CSS:** Використання Flexbox/Grid для ідеальної сітки товарів.
* **Axios:** Надійна обробка запитів до нашого API.

### ⚙️ Backend (Серверна частина)
* **Node.js & Express:** Потужна архітектура для обробки логіки магазину.
* **Railway Hosting:** Стабільна робота сервера 24/7 без потреби тримати комп'ютер увімкненим.
* **Custom API:** Власна розробка ендпоінтів для отримання даних про товари.
